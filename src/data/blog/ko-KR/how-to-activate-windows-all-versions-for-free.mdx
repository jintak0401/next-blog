## 下載工具

<Alert icon='🔗' title='連結' color='blue' radius='md'>
  [下載 CMWTAT
  Digital](https://github.com/TGSAN/CMWTAT_Digital_Edition/releases)
</Alert>

1. 點擊 CMWTAT_Digital_Release_2_5_0_0.exe 下載工具。

![下載 CMWTAT Digital](</static/images/blog/how-to-activate-windows-all-versions-for-free/step(1).png>)

2. 打開工具後選擇你的 Windows 版本，這裡我是專業版，因此選擇 `Professional`。
3. 按下 `Activate`

![使用 CMWTAT Digital](</static/images/blog/how-to-activate-windows-all-versions-for-free/step(2-3).png>)

稍等片刻

![移除舊密鑰](</static/images/blog/how-to-activate-windows-all-versions-for-free/activating(1).png>)

![取得權限](</static/images/blog/how-to-activate-windows-all-versions-for-free/activating(2).png>)

![啟用中](</static/images/blog/how-to-activate-windows-all-versions-for-free/activating(3).png>)

成功啟用

![成功啟用](</static/images/blog/how-to-activate-windows-all-versions-for-free/activating(4).png>)

## 查看啟用狀態

到 Windows 設定的啟用頁面，就會看到 Windows 已啟用。
